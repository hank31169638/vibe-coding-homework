# 糖果消消樂遊戲開發說明文件

## 📋 專案概述

這是一個使用純HTML、CSS、JavaScript開發的糖果消除遊戲，類似經典的Candy Crush遊戲。遊戲採用8x8網格，玩家通過點擊選擇糖果並與相鄰糖果交換來產生三消效果。
